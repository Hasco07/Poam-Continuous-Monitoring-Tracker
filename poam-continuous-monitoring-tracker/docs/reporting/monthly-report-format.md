# Monthly report format

The monthly report is meant to be readable by:
- ISSM/ISSO leadership
- system owners / engineering leadership
- assessors during inspection readiness reviews

## Recommended sections
1. Reporting period
2. Executive summary (top risks, what changed, asks/blockers)
3. Metrics (counts by severity/status, overdue aging)
4. Top overdue or high-risk items (table)
5. Newly opened this period (table)
6. Closed this period (table)
7. Notes / constraints / exceptions (if any)
